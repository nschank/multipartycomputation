Learning Center

In the interest of making it easier for anyone to learn about Multiparty Computation, we plan to write about and explain the basics as much as possible. As such, we have created a comprehensive learning center encompassing the entirety of Multiparty Computation, for anyone to jump into. It is formatted similarly to an online textbook; this page is our table of contents. Start wherever you feel comfortable!

Background

These lessons try to assume as little background as possible, but there is still some necessary! In order to understand this book, you need some background in:

  • Basic probability
  • Boolean gates
  • Logarithms and modular arithmetic
  • Big-Oh Notation
  • Experiment Notation
  • Some specific schemes

If you've never seen these things before, or need a refresher, there's a cheat sheet available to review or keep open in another tab.
