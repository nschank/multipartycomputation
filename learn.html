Learning Centre

In the interest of making it easier for anyone to learn about Multiparty Computation, the main focus of this project is writing about and explaining the basics as much as possible. As such, we have created a comprehensive learning centre encompassing the entirety of Multiparty Computation, for anyone to jump into. It is formatted similarly to an online textbook; a table of contents follows. Start wherever you feel comfortable!

Background

These lessons try to assume as little background as possible, but there is still some necessary! In order to understand this book, you need some background in:

  • Basic probability
  • Logarithms and modular arithmetic
  • Basic coding and scripting
